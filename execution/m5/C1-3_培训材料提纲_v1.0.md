# C1-3 培训材料提纲（v1.0）

## 1. 培训目标
- 让业务用户可独立完成日常研判、导出与上报流程。
- 让运维用户可独立完成部署、巡检、灰度与回滚演练。

## 2. 课程结构
1. 系统总览与角色权限（15分钟）
2. 业务操作实训（30分钟）
3. 导出与报告标准（15分钟）
4. 运维脚本与巡检（25分钟）
5. 故障演练与回滚（25分钟）
6. Q&A 与考核（10分钟）

## 3. 实操清单
- 用户侧：
  - 登录并查看预测、事件、归因
  - 执行 PNG/CSV/Markdown 导出
- 运维侧：
  - 执行 deploy/smoke/monitor
  - 执行 gray drill 与 runbook drill
  - 提交模型 rollback 任务并验证

## 4. 培训交付物
- 课件（PPT）
- 操作手册（本阶段 C1-3 主文档）
- 演练记录（签到、通过率、问题清单）

## 5. 通过标准
- 业务用户：关键操作通过率 >= 90%
- 运维用户：演练脚本执行通过率 = 100%
