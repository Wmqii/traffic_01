2. **启动服务并命中两个接口，确保返回 200**

   ```powershell
   cd D:\小桌面\交通拥堵
   $env:PYTHONPATH = "."
   $env:PYTHONIOENCODING = "utf-8"
   python -c "import os, subprocess, time, urllib.request; cmd=['python','-m','uvicorn','backend.app:app','--port','8082','--log-level','warning']; env=os.environ.copy(); env.update({'PYTHONPATH':'.','PYTHONIOENCODING':'utf-8'}); proc=subprocess.Popen(cmd, env=env); time.sleep(3); urls=['http://127.0.0.1:8082/api/v1/congestion/events','http://127.0.0.1:8082/api/v1/predictions/segments/SEG-4001'];\ntry:\n    for url in urls:\n        req=urllib.request.Request(url)\n        with urllib.request.urlopen(req, timeout=10) as resp:\n            body=resp.read(200).decode('utf-8').replace('\\n',' ')\n            print(url, resp.status, body[:120])\nfinally:\n    proc.terminate()\n    proc.wait(10)"
   ```

   输出：
   ```
   http://127.0.0.1:8082/api/v1/congestion/events 200 [{"event_id":"event-bridge",...}]
   http://127.0.0.1:8082/api/v1/predictions/segments/SEG-4001 200 {"segment_id":"SEG-4001",...}
   ```

   说明：两个目标接口均返回 200，且响应包含当期对齐事件与分段预测数据。
