## 4. 自动化方案设计

### 4.1 执行框架
- 使用 `FastAPI TestClient` 直连 `backend.app`，不依赖外部服务启动。
- 单脚本串行执行检查项，输出统一 JSON 结果并以进程退出码标识通过/失败。

### 4.2 覆盖域
- 鉴权与登录：无效登录、viewer/analyst/admin 登录与 token 有效性。
- 核心查询：预测、拥堵事件、归因接口可用性。
- RBAC：viewer 对 analyst/admin 接口的禁止访问校验。
- 缓存能力：热力图层命中、缓存统计、缓存刷新。
- 异步任务：提交、轮询、列表、不存在任务分支。
- 审计接口：admin 审计可用与 endpoint_count 合理性。
