## 2. 项目章程

### 2.1 项目名称
交通流量预测与拥堵成因分析系统

### 2.2 项目目标
- G1：实现 15/30/60 分钟短期交通预测。
- G2：实现拥堵事件识别与成因归因。
- G3：实现 WebGIS 可视化与多维分析。
- G4：形成可运维、可审计、可持续迭代的工程体系。

### 2.3 范围边界
- 范围内：数据接入治理、预测建模、归因分析、API、可视化、权限与审计。
- 范围外：信号灯闭环控制、移动端原生 App、商业数据采购。

### 2.4 干系人
- 业务方：交通管理、城市运营
- 执行方：PM/Data/Algo/Backend/Frontend/Test/Ops/Doc Agents
- 验收方：业务负责人 + 技术负责人 + QA 负责人
