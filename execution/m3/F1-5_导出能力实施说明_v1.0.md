## 2. 实施目标
- 在前端新增“导出中心”，支持业务展示数据的离线导出与汇报复用。
- 输出图表截图（PNG）、结构化明细（CSV）和文本简报（Markdown）。

## 4. 实施内容

### 4.1 导出中心 UI
- 在“管理”页新增导出卡片，包含：
  - 图表下拉选择器 + PNG 导出按钮
  - 4类 CSV 导出按钮
  - 简报 Markdown 导出按钮

### 4.2 图表 PNG 导出
- 使用 ECharts `getDataURL` 导出指定图表。
- 支持图表：`trendChart`、`waterfallChart`、`boxplotChart`、`sankeyChart`、`heatReplayChart`。

### 4.3 CSV 导出
- 事件列表 CSV（事件 ID、路段、等级、置信度、时间窗）
- 热力数据 CSV（路段、网格、热力分、窗口标签）
- 预测特征 CSV（路段、预测等级、置信度、特征名、值、来源）
- 归因驱动 CSV（事件、路段、严重度、驱动项、影响度）

### 4.4 简报导出
- 导出 Markdown 报告，内容包括：
  - 当前筛选条件（时间窗、路段、事件）
  - 核心指标（预测等级、置信度、活跃事件数、最高热力分）
  - 关键结论与建议动作
  - 生成时间戳

## 5. 实施结果
- 纯前端实现，无新增重依赖。
- 导出动作写入运行日志，便于审计留痕。
- 导出能力与现有联动逻辑兼容。
