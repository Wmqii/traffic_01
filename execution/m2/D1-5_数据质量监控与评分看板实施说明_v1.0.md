## 4. 评分模型

### 4.1 维度
- 完整性：`(total - missing) / total`
- 有效性：`(total - anomaly) / total`
- 唯一性：`(total - duplicate) / total`
- 时效性：`(total - delay) / total`
- 管道一致性：`(alignment_score + feature_coverage_score) / 2`

### 4.2 权重
- completeness: 0.2
- validity: 0.2
- uniqueness: 0.2
- timeliness: 0.2
- pipeline_consistency: 0.2

### 4.3 阈值
- PASS 阈值：`>= 90`
- 评分等级：A/B/C/D
